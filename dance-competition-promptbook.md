## 1. Event Overview — How to Use This Book

**Goal:** Explain the sections of the prompt book so anyone on the team can navigate it.

**Logic:** The book must teach itself. A new volunteer or returning team member should open this page, understand what each section contains and when to use it, without needing you there to explain. If the book only makes sense to the person who wrote it, it fails the moment that person is busy putting out fires.

## 4. Policies to Print and Post

**Goal:** Have every enforceable rule on paper and physically posted at the venue before the event starts.

**Logic:** A policy settles disputes only if it exists *before* the dispute. Post each where its problem happens — backstage rules at the backstage door, music rules at check-in. A posted policy is enforcement; a verbal one is a debate.

**Policies to Draft (then print copies):**
- [ ] **Backstage Rules:** No parents backstage. Dancers only. Quiet zones. No phones during performance.
- [ ] **Music Rules:** Accepted formats. No profanity. No unlicensed tracks. Late submission penalty.
- [ ] **Competitor Conduct:** Dress code. Check-in deadlines. Judging protest procedure.
- [ ] **Audience Rules:** No flash photography. No moving during acts. No saving seats.
- [ ] **Lost Child / Lost Item Protocol:** Where to go. Who to ask.
- [ ] **Emergency Procedures:** Fire exits. Assembly point. Medical station.

**Posting Location Map:**

| Policy | Location(s) |
|---|---|
| Backstage Rules | Backstage entrance, green room, dressing rooms |
| Music Rules | Check-in table, audio booth |
| Competitor Conduct | Registration desk, backstage bulletin board |
| Audience Rules | Box office, house doors, program insert |
| Lost Child / Lost Item | Box office, front-of-house table |
| Emergency Procedures | Every exit, green room, lobby |

**Goal:** One intake flow that catches errors early: enforce policies, collect titles, draw order numbers, collect music from captains, and set a deadline that compresses last-minute changes.

**Logic:** An error caught at intake costs two minutes — caught at order check it stalls a line. Chasing dozens of individual competitors is chaos; chasing ~10 captains is manageable. The deadline doesn't stop changes, it forces bulk submission early so order check is verification, not data entry.

**Goal:** Define the complete judge table setup: position, tablecloths, lighting, notation folders, pencils, snacks.

## 14. Judge Review Sheets and Decision

**Goal:** Provide the sheets judges write on and the structure for how their decision is recorded.

**Logic:** Sheet design determines tabulation speed. Pre-printing competitor names in act order means judges write almost nothing. Every decision exists on paper, defensible later.

**Judge Scoring Sheet (per judge, per category — print with competitor names pre-filled):**

```
PRELIMINARY / SEMI / FINAL (circle one)
Category: ___________  Judge: ___________  Date: ___________

Act# │ Competitor │ Tech/10 │ Artis./10 │ Perf./10 │ Diff./10 │ Total/40
─────┼────────────┼─────────┼───────────┼──────────┼──────────┼─────────
     │            │         │           │          │          │
     │            │         │           │          │          │
     │            │         │           │          │          │
     │            │         │           │          │          │
     │            │         │           │          │          │
     │            │         │           │          │          │
     │            │         │           │          │          │
     │            │         │           │          │          │

Notes: ______________________________________________________

Judge Signature: __________________
```

**Rubric:** Technical /10 (execution, alignment, control) / Artistry /10 (musicality, expression) / Performance /10 (stage presence, energy) / Difficulty /10 (choreography complexity, risk)

**Tabulation Flow:**
1. Runners collect all sheets → deliver to tabulation table (secure, quiet)
2. Enter scores per competitor (drop high/low if used, average rest)
3. Rank by total → determine advancing number
4. Double-check top 3 against raw sheets
5. Seal in envelope → hand to emcee / SM
