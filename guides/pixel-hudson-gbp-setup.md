## Step 8: Verification — Request Postcard

1. On the verification screen, select **"Postcard"** as the verification method
2. Confirm the mailing address is **185 Grange Rd Ste 2, Otisville, NY 10963**
3. Click **"Request postcard"** to trigger the mail

### Important — While Waiting for the Postcard

- [ ] **Do NOT** edit the business name, address, or category — any change will invalidate the code
- [ ] **Do NOT** request a second postcard — this cancels the first one and restarts the clock
- [ ] **Do NOT** add a phone number or website during this period
- [ ] Postcard arrival: **5–14 business days** (domestic US)
- [ ] Verification code expires: **30 days** from the request date

### If Postcard Does Not Arrive

- Wait the **full 14 business days** before taking any action
- After 14 business days, log back in, and request a replacement postcard
- If the option is not available, contact Google Business Profile support

## Troubleshooting

### Google offers video verification instead of postcard

Google may auto-select video verification for service-area businesses. If postcard is not offered:
- Complete the video verification if available (it's faster — 1–5 business days)
- If postcard is required, contact Google Business Profile support to request it

### "This business already exists" error

If Google shows an existing profile at 185 Grange Rd:
- Click **"Claim this business"** instead of creating a new one
- Do NOT create a duplicate — Google will suspend both

### Verification code expired

If the 30-day window passes before entering the code:
- Request a new postcard
- The old code is invalid — do not attempt to enter it
