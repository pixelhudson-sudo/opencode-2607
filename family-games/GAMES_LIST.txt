HOW TO ADD ANY NEW GAME (for AI / yourself)
================================================================================

The system is designed for easy expansion. To add a new game:

1. Create `games/your_game.py` following BaseGame template (see games/snake.py)
2. In main.py, add one line:
   from games.your_game import YourGameClass
   register_game('your_game_id', YourGameClass,
       name='Your Game',
       description='What the game is about',
       category='Category Name',
       min_players=1, max_players=3,
       icon='🎮')

The 4 methods to implement:
  - __init__(self, players, engine): Setup. Store engine.fonts['small'] etc.
  - handle_event(self, event): pygame events (keys, etc.)
  - update(self, dt): Game logic. dt = seconds since last frame.
  - render(self, screen): Draw everything to screen surface.

When finished: set self.finished = True and self.results = list of dicts:
  [{'player': Player, 'score': int, 'winner': bool}]

That's it! The engine handles menus, leaderboards, badges, and persistence.
